# Supply Chain Analysis (Power BI)

## 📌 Quick Summary

Power BI dashboard analyzing supply chain performance across sales, inventory, delivery, and quality to identify inefficiencies and improve decision-making.

---

## 📊 Project Overview

This project analyzes supply chain performance across key areas such as sales, inventory, delivery, and product quality. The objective is to identify inefficiencies and provide actionable business recommendations.

An interactive Power BI dashboard was developed to monitor and visualize key supply chain metrics.

## 🛠 Tools & Technologies

* Power BI
* DAX
* CSV Dataset
* phpMyAdmin

## 📂 Data Source

The dataset was extracted from a MySQL database using phpMyAdmin and converted into CSV format for analysis.

## 📊 Key Metrics

* Total Revenue: 577.60K
* Total Products Sold: 46K+
* Average Shipping Time
* Average Defect Rate

## 🔍 Key Insights

* Skincare products generate the highest revenue and sales
* Cosmetics category has higher shipping time, indicating delivery delays
* Inventory is mostly well-managed, but 24% of products are low stock
* Supplier performance varies, with Supplier B performing best
* Higher manufacturing cost does not guarantee better product quality

## 📌 Features

* Multi-page dashboard (Overview, Sales, Inventory, Delivery, Quality)
* KPI cards for performance tracking
* Stock status classification (Low Stock / Sufficient Stock)
* Supplier performance and defect rate analysis
* Logistics analysis (transport modes and shipping cost)

## ⚠️ Challenges Faced

* Supplier naming inconsistency across datasets (Supplier 1–5 vs Supplier A–C)
* Difficulty in establishing relationships between tables
* Initial challenges in creating DAX measures
* Designing an effective and user-friendly dashboard

This project demonstrates how data analytics and visualization can improve supply chain efficiency, optimize inventory management, and enhance business decision-making.
